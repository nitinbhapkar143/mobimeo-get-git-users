
# Mobimeo - Git Users
API endpoint to search for GitHub users by the programming language they use in their public repositories


## Installation

Clone the repo
Install dependencies.

```bash
npm install
```
Run app in dev
```bash
npm run dev
```
Run test
```bash
npm test
``` 
Build Docker image
```bash
docker build -t express-typescript .
```  
Run docker image
```bash
docker run -p 8000:8000 express-ts
``` 
Docker compose
```bash
docker-compose up
```
